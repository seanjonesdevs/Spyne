# Spyne

Spyne is a phone number lookup and spam detection website built to help users quickly identify unknown callers and potential spam numbers.

## Overview

This project was created as an experimental web application to explore phone number search functionality, spam detection workflows, and modern frontend development.

The goal was to build a clean and simple platform where users could search phone numbers and view information about potentially suspicious or unwanted callers.

## Features

- Phone number search interface
- Spam detection concept
- Clean responsive UI
- Web-based lookup experience

## Tech Stack

- Next.js
- TypeScript
- React
- Vercel

## Status

Personal side project / experimental build. Development was started but not fully completed.
